# CellUniverse Team Page

Static GitHub Pages page for CellUniverse namecards and the project repository block.

## Edit Content

- Update people in `script.js`.
- Put real portrait image files in this repo and set each person's `photo` field to the file path, for example `assets/ada-chen.jpg`.
- Update the repository link in `index.html`.

## Deploy To GitHub Pages

1. Push this folder to a GitHub repository.
2. Open repository `Settings` -> `Pages`.
3. Choose `Deploy from a branch`.
4. Select the branch and `/ (root)`.
5. Save.

GitHub Pages can serve this directly; no build step is required.
